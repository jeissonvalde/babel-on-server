  Tutorial mongodb

  Font Awesome (libre)
    Api que me ofrece íconos para mi web, diseñados y de uso libre.
    http://fontawesome.io/

  HTML 2 Jade (libre)
    Me permite cambiar el código de un estilo a otro:
    html --> jade
    Usa por ejemplo con pug.
    http://html2jade.org/

  Moment.js (libre)
    Me realiza operaciones con datos de fechas y formatea para dar al cliente textos legibles
    http://momentjs.com/

  Superagent Y axios (libre)
    APIs para hacer requests. se caracteriza por su forma de escritura, es otra opción similar a JQuery.
    Superagent --> https://github.com/visionmedia/superagent/branches
    Axios --> https://github.com/mzabriskie/axios

    Certificado para que mi web tenga un https
    https://letsencrypt.org/

  Luke Hass projects (libre)

  Live Reload (libre)
